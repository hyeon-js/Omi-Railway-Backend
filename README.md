## 사용법?
- 아직 미완성이나, 다음 방법을 통해 실행은 가능
```sh
$ git clone https://github.com/hyeon-js/Omi-Railway-Backend
$ cd Omi-Railway-Backend
$ ./gradlew bootRun
```
- `http://localhost:8080/omi?line=${노선명}` 등으로 요청 가능
